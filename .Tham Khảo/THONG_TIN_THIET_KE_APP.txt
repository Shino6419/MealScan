THONG TIN THIET KE APP MEALSCAN
Cap nhat: 12/05/2026

1. Muc tieu giao dien

MealScan la app React Native ho tro nguoi dung theo doi calo moi ngay.
App co cac chuc nang chinh:
- Dang nhap, dang ky, quen mat khau.
- Chon/chup anh mon an.
- AI nhan dien mon an va uoc tinh calo.
- Luu mon an vao lich su an uong.
- Thong ke calo theo ngay/tuan.
- Cai dat tai khoan, thong tin the trang va thong bao.

Giao dien can tao cam giac sach, hien dai, de doc, phu hop voi ung dung suc khoe/an uong.

2. Combo mau chinh: Mint Blue

Nen man hinh:
- #F0FDFA

Mau chinh:
- #14B8A6

Mau phu / diem nhan:
- #38BDF8

Mau the/card:
- #FFFFFF

Vien nhe:
- #CCFBF1

Chu chinh:
- #0F172A

Chu phu:
- #64748B

Chu placeholder / inactive:
- #94A3B8

Mau canh bao:
- #F97316

Quy tac:
- Khong nen dung qua nhieu mau ngoai combo tren.
- Button chinh dung #14B8A6.
- Icon hoac stripe trang tri co the dung #38BDF8.
- Nen tong the nen sang, khong dung nen toi lam mau chu dao.

3. Font va chu

Hien tai dung font mac dinh cua React Native.

Quy tac kich thuoc:
- Tieu de lon cua header: 30-34px, fontWeight 800.
- Kicker nho tren header: 12-13px, fontWeight 800, mau #14B8A6.
- Subtitle: 15-16px, lineHeight 22-24, mau #64748B.
- Label form: 14px, fontWeight 700-800.
- Text button: 15-16px, fontWeight 800.

Quy tac:
- Khong scale font theo viewport.
- Letter spacing giu 0.
- Text trong button phai vua khung tren man hinh nho.

4. Header / Headbar thong nhat

Tat ca man hinh chinh nen co header dang card trang:
- Nen #FFFFFF.
- Border #CCFBF1.
- Border radius 8.
- Co stripe tren cung mau #38BDF8, cao 5px.
- Ben trai gom kicker, title.
- Ben phai la icon trong o vuong nen #ECFEFF, radius 8.
- Subtitle nam duoi title.

Vi du noi dung:
- Thong ke: kicker "Today", title "Thong ke"
- Chup anh: kicker "AI scan", title "Chup anh"
- Ket qua AI: kicker "AI result", title "Ket qua AI"
- Cai dat: kicker "Settings", title "Cai dat"

Quy tac:
- Header khong dung navigation header mac dinh cua Expo Router.
- Stack/Tab header nen de headerShown: false.
- Khong de title mac dinh hien tren cung man hinh.

5. Layout tong the

App dung bottom tab navigation voi 3 tab:
- Thong ke
- Chup anh
- Cai dat

Man hinh thong ke:
- Dung layout dang dashboard/timeline.
- Co header thong nhat.
- Co thanh chon ngay/tuan.
- Phan mon an theo ngay nen nam phia tren theo lich.
- Co thong ke tong calo trong ngay.
- Co danh sach bua an theo timeline.
- Co lich de chon ngay xem chi tiet.

Man hinh chup anh:
- Khong dung scroll neu noi dung vua man hinh.
- Vung camera/anh co ti le 1:1.
- Nut chinh: tiep tuc xem ket qua.
- Nut phu: chon anh tu thu vien / chon anh khac.
- Khi da co anh, hien anh trong khung vuong.

Man hinh ket qua AI:
- Hien anh mon an da chon.
- Hien trang thai model:
  - Dang phan tich mon an.
  - Du doan thanh cong.
  - Loi khi chay model.
- Tu dien ten mon an va calo uoc tinh neu AI tra ket qua.
- Cho phep nguoi dung sua ten mon va calo truoc khi luu.
- Co chon loai bua an: Sang, Trua, Toi, An nhe.

Man hinh cai dat:
- Khong dung scroll neu noi dung vua man hinh.
- Co profile card o dau.
- Cac nut cai dat hien trong cung mot trang:
  - Tuy chon tai khoan.
  - Thong tin the trang.
  - Cai dat thong bao.
  - Ngon ngu.
  - Dang xuat.

Man hinh auth:
- Dang nhap, dang ky, quen mat khau.
- Khong hien Expo header mac dinh.
- Co panel/headbar theo style Mint Blue.
- Form input ro rang, de bam tren mobile.

6. Component pattern

Button chinh:
- Background #14B8A6.
- Text #FFFFFF.
- Border radius 8.
- Min height 52-54.
- Co icon Ionicons neu phu hop.

Button phu:
- Background #FFFFFF.
- Border #CCFBF1.
- Text #14B8A6.
- Border radius 8.

Card:
- Background #FFFFFF.
- Border #CCFBF1 hoac #E2E8F0.
- Border radius 8.
- Shadow nhe, khong qua dam.
- Khong long card trong card neu khong can thiet.

Input:
- Background #FFFFFF.
- Border #E2E8F0.
- Text #0F172A.
- Placeholder #94A3B8.
- Min height 52-54.
- Border radius 8.

Segment control:
- Dung cho chon loai bua an.
- Item active: background #14B8A6, text #FFFFFF.
- Item inactive: background #FFFFFF, border #E2E8F0, text #475569.

7. Icon

Dang dung Ionicons tu @expo/vector-icons.

Goi y icon:
- Thong ke: analytics-outline / bar-chart-outline.
- Chup anh: camera-outline / scan-outline.
- Ket qua AI: sparkles-outline.
- Profile/cai dat: settings-outline / person-outline.
- Luu: checkmark-circle-outline.
- Anh: image-outline.
- Canh bao: warning-outline.

Quy tac:
- Button nen co icon khi hanh dong ro rang.
- Icon trong header dat trong o vuong nen #ECFEFF.

8. Khoang cach va bo cuc

Screen padding:
- Horizontal: 20-24.
- Top: 48-56.
- Bottom: 22-32.

Khoang cach:
- Giua header va noi dung: 14-22.
- Giua cac card: 12-16.
- Padding trong card: 14-20.

Border radius:
- Mac dinh 8.
- Khong dung radius qua lon neu khong co ly do.

9. Quy tac mau cho trang thai

Thanh cong / active:
- #14B8A6

Thong tin / highlight:
- #38BDF8

Canh bao:
- #F97316

Nen thong bao nhe:
- #ECFEFF

Loi:
- Co the dung #EF4444 neu can hien thi loi nghiem trong.

10. Model AI va man hinh du doan

Thu muc model:
- assets/models/best_int8.tflite
- assets/models/labels.txt

Thong tin model:
- YOLOv8n-cls da convert sang TFLite int8.
- Input image size: 224x224.
- So class: 30.
- labels.txt phai giu dung thu tu class khi train.

Quy tac UI khi chay AI:
- Luon hien trang thai dang phan tich khi model dang xu ly.
- Neu model loi, hien loi ro rang nhung van cho nguoi dung nhap thu cong.
- Ket qua AI chi la uoc tinh, nguoi dung duoc sua truoc khi luu.

11. Nguyen tac phat trien man hinh moi

Khi tao man hinh moi:
- Dung background #F0FDFA.
- Dung header thong nhat neu la man hinh chinh.
- Khong dung mau lech khoi Mint Blue neu khong co ly do.
- Khong bat scroll neu noi dung co the vua man hinh.
- Neu noi dung dai, dung ScrollView voi showsVerticalScrollIndicator=false.
- Button, input, card phai theo pattern trong file nay.
- Kiem tra UI tren man hinh dien thoai nho.
- Kiem tra text khong bi tran, khong de component che nhau.

12. Ten man hinh hien tai

Auth:
- app/(auth)/login.tsx
- app/(auth)/register.tsx
- app/(auth)/forgot-password.tsx

Tabs:
- app/(tabs)/index.tsx: Thong ke
- app/(tabs)/camera.tsx: Chup anh
- app/(tabs)/profile.tsx: Cai dat

Khac:
- app/meal-result.tsx: Ket qua AI
- app/meal-detail.tsx: Chi tiet bua an

13. Ghi chu build native

Vi app dung native module de chay TFLite:
- Khong chay duoc day du trong Expo Go.
- Can dev build bang npx expo run:android.
- Khi them/sua native package, can build lai native app.

Tren Windows nen dat project o duong dan ngan, vi React Native New Architecture co the sinh path C++ rat dai.
Vi du nen dung:
- C:\MealScan

Khong nen dung path dai nhu:
- C:\Users\<user>\Documents\Visual Studio Code\React+\MealScan
